Class Detail
Caption(opt_options)
Creates a new Caption object. Use captions to communicate short messages to users like a title or simple instructions like 'click for more particles'.
Parameters:
{Object} opt_options Optional
Options.
{string} opt_options.position Optional, Default: 'top left'
A text representation of the caption's location. Possible values are 'top left', 'top center', 'top right', 'bottom left', 'bottom center', 'bottom right', 'center'.
{string} opt_options.text Optional, Default: ''
The caption's text.
{number} opt_options.opacity Optional, Default: 0.75
The caption's opacity.
{string} opt_options.borderWidth Optional, Default: '1px'
The caption's border width.
{string} opt_options.borderStyle Optional, Default: 'solid'
The caption's border style.
{Array|string} opt_options.borderColor Optional, Default: 0.75
The caption's border color.
