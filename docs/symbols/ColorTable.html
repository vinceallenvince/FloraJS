Class Detail
ColorTable()
Creates a new ColorTable. Use a color table to create a map of keywords to color ranges. Instead of manually passing start and end colors when creating color palettes, you can use getColor() and pass a keyword to receive the start and end colors. 
var heat = defaultColors.getColor('heat');
console.log(heat.startColor); // -> [255, 132, 86]
console.log(heat.endColor); // -> [175, 47, 0]

{Object|Array} getColor(options, startColor, endColor)
Returns start and end colors from a key in the color table. 
var heat = myColorTable.getColor('heat');
console.log(heat.startColor); // -> [255, 132, 86]
console.log(heat.endColor); // -> [175, 47, 0]

var heat = myColorTable.getColor('heat', true);
console.log(heat); // -> [255, 132, 86]

var heat = myColorTable.getColor('heat', false, true);
console.log(heat); // -> [175, 47, 0]
Parameters:
{Object} options
A set of options. Required: options.name {number} The name of the entry in the color table. Optional: options.startColor {boolean} Pass true to only return the start color. options.endColor {boolean} Pass true to only return the end color.
{Array} startColor
An array representing the start color. ex: [255, 100, 50].
{Array} endColor
An array representing the end color. ex: [155, 50, 10].
Returns:
{Object|Array} Either an object with startColor and endColor properties or an array representing a start or end color.
